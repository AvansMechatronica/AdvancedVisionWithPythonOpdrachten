# Object detectie

In deze sectie behandelen we technieken voor het detecteren en lokaliseren van objecten binnen afbeeldingen of video's. Object detectie is een fundamenteel onderdeel van veel computer vision-toepassingen, zoals gezichtsherkenning, autonome voertuigen en beveiligingssystemen. We zullen verschillende algoritmen bespreken die worden gebruikt voor object detectie, zoals Haar Cascades, HOG (Histogram of Oriented Gradients) en moderne deep learning-gebaseerde methoden zoals YOLO (You Only Look Once) en SSD (Single Shot MultiBox Detector), evenals hun implementaties met behulp van de OpenCV- en TensorFlow-bibliotheken.
